# Avaliador de Senha

O projeto avalia a senha digitada pelo usuário em um campo texto e retorna a informação como uma das possíveis avaliações abaixo:

- MUITO CURTA
- MUITO FRACA
- FRACA 
- BOA 
- FORTE 
- MUITO FORTE

### Tecnologias Utilizadas

- Java 8
- Spring Framework 4.1.5
- JSF 2.2.10
- JUnit 3.8.1
- Primefaces 5.1
- Bootstrap 3.3.4
